### 教师信息

#### 查询

无法得到其它教师的ID

post http://localhost:8081/teacher/queryWithoutID

请求格式：

```
?pageIndex=1&pageSize=2
```

```json
{
    "teacherName":null,
    "facultyCode":"0001",
    "facultyName": null
}
```

返回格式:

```json
{
    "code": "0",
    "message": "查询成功",
    "data": {
        "total": 1,
        "pageIndex": 1,
        "tableData": [
            {
                "teacherName": "李四",
                "facultyCode": "0001",
                "facultyName": "信息科学与工程学院"
            }
        ]
    }
}
```
